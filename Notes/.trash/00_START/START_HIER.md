## 💬 HOE WERKT HET UPDATE SYSTEEM?

### Jij praat, ik typ:
1. **Pak een template** uit `Templates/` folder
2. **Zet cursor op de "..."** en praat
3. **Stuur naar mij** (Claude)
4. **Ik update automatisch**:
   - Daily update file
   - TODO.md status
   - Relevante guides
   - Progress tracking

